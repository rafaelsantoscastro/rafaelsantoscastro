- Criação de Dashboards Básicos (Power BI/Excel):
Desenvolva dashboards simples no Power BI ou Excel, exibindo métricas básicas de desempenho de vendas ou dados financeiros, fornecendo uma visão geral dos dados.
- Modelo de Regressão Simples (R/Python - Scikit-learn):
Construa um modelo básico de regressão linear para prever vendas ou qualquer outra métrica relevante usando bibliotecas de machine learning como Scikit-learn em Python ou R.

Crie um banco de dados simples usando MySQL ou PostgreSQL, e pratique consultas básicas para extrair e manipular informações para relatórios de vendas ou inventário.
